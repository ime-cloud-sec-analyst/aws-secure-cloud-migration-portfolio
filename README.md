# AWS Secure Cloud Migration Portfolio

This repository demonstrates practical AWS migration architecture and security design aligned with the responsibilities of a **Lead Technical Architect / Cloud Security Architect**.

## Portfolio Overview

This portfolio simulates real-world enterprise migration scenarios from **on-premise infrastructure to AWS**, focusing on secure architecture design, migration strategy, and enterprise cloud governance.

Key architectural considerations include:

- Migration readiness assessment
- System dependency mapping
- Security risk evaluation
- Migration strategy selection
- Secure AWS target architecture design
- Cloud governance and operational readiness

---

## Labs Included

### Lab 1 — Secure Cloud Migration Readiness Assessment

This lab demonstrates how a **Lead Technical Architect** evaluates an organisation before migrating workloads from on-premise infrastructure to AWS.

#### Capabilities Demonstrated

- Workload discovery and inventory
- System dependency analysis
- Security risk identification
- Migration strategy selection (Rehost, Replatform, Refactor)
- Mapping on-prem workloads to AWS services
- Migration planning using AWS migration services
