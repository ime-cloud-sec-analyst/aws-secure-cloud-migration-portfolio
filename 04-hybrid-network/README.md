# Lab 4 — Hybrid Network Architecture

## Overview

This lab demonstrates how enterprises securely connect on-premise infrastructure to AWS using a hybrid networking architecture.

Many organisations cannot move everything to the cloud immediately. Hybrid networking allows on-premise systems and cloud resources to operate together during migration.

This architecture ensures secure connectivity, controlled network routing, and isolation of sensitive workloads.

---

## Objective

The objective of this lab is to design a secure hybrid network architecture connecting an on-premise data centre with AWS infrastructure.

This design enables organisations to migrate workloads gradually while maintaining operational continuity.

---

## Architecture Scenario

An enterprise organisation operates a traditional on-premise data centre hosting:

- Active Directory services
- Internal business applications
- Database systems
- File storage

As part of a cloud migration strategy, the organisation deploys workloads to AWS while maintaining connectivity with on-premise infrastructure.

---

## Hybrid Network Architecture

The hybrid architecture connects the corporate network to AWS using secure connectivity services.
